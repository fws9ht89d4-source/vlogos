# Vlog OS

A small personal web app for planning iPhone vlogs. It saves your board,
workflow settings, and shot checklist to Supabase using your private access
code.

## Local links

- Mac: http://localhost:5173/index.html
- iPhone on same Wi-Fi: http://192.168.50.94:5173/index.html

## Free anywhere link

Deploy this folder to Vercel. Vercel will give you a free address like:

```text
https://vlog-os.vercel.app
```

That Vercel address works from your iPhone and Mac anywhere, even away from
home Wi-Fi.

## Deploy steps

1. Make a free Vercel account at https://vercel.com
2. Put this folder in a GitHub repo.
3. In Vercel, choose Add New Project.
4. Pick the GitHub repo.
5. Use the default settings and deploy.
6. Open the Vercel link on iPhone.
7. Add that Vercel link to the Home Screen.

After this, you should use the Vercel link instead of the local Wi-Fi link.
